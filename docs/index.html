기술 스택 & 개발 정보

Enhanced Input — 코드 생성

InputMappingContext / InputAction을 .uasset 없이 PlayerController에서 NewObject + MapKey로 구성. 입력 에셋 0개.

Canvas HUD — 순수 C++

UMG 위젯 없이 AHUD::DrawHUD() Canvas 드로잉으로 자원·HP 바·웨이브·상점·포탑 조준점까지 전부 구현.

상태머신 게임 루프

PreWave→Gather→Combat→Shop→Victory/GameOver 7상태 사이클을 GameMode가 관리. 웨이브 스폰 소프트웨어 자원 카운트 통합.

IDamageable 인터페이스

블록·몬스터·기지가 공통 TakeHit(Damage, AttackerPower)를 구현. 라인트레이스가 맞은 액터를 인터페이스 캐스트로 피해 전달.

Dynamic Material Instance

네온 부모 머티리얼 1개만 수동 제작. 색·발광·피격 플래시는 전부 C++ MID 파라미터로 실시간 제어.

Niagara 트레일 & 이펙트

기체·몬스터 파티클 트레일, 카메라 쉐이크, 네온 펄스 등 시각 피드백을 코드 측에서 바인딩 호출.
